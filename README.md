# Website UMKM Desa Gedung Harapan

Website ini dibuat untuk membangun UMKM Desa Gedung Harapan.

Anda bisa berkontribusi dengan clone website ini:

```
git clone <url>
cd tubes-kapsel-gdharapan
npm install
npm run dev
```
